Osm data getter
---------------

The module is designed to retrieve data from OSM.

.. code-block:: python

    import warnings
    import os
    import sys

    warnings.simplefilter("ignore")
    current_dir = os.getcwd()

    project_root = current_dir.split("\\examples")[0]

    sys.path.append(project_root)

    from factfinder.src.data_getter import GeoDataGetter
    from factfinder.src.constants import OSM_TAGS

    geo_data_getter = GeoDataGetter()

    tdict = [(category, category_tags) for category, category_tags in OSM_TAGS.items()]
    test_gdf = geo_data_getter.get_features_from_id(osm_id=421007, tags={tdict[0][0]: tdict[0][1]})
    test_gdf
